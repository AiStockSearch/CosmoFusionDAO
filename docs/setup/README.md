# 🔧 Настройка и конфигурация

Эта папка содержит документацию по настройке и конфигурации проекта CosmoFusion DAO.

## 📁 Файлы

### 🌐 **Переменные окружения**
- [`ENV_EXAMPLE.md`](./ENV_EXAMPLE.md) - Полный пример файла .env с комментариями
  - Telegram Bot настройки
  - Ссылки на социальные сети
  - Аналитика (Amplitude, Firebase, Sentry)
  - Инструкции по получению ключей

### 🔒 **Безопасность**
- [`SECURITY_SETUP.md`](./SECURITY_SETUP.md) - Настройка безопасности проекта
  - CSP (Content Security Policy)
  - Заголовки безопасности
  - Аудит зависимостей
  - Мониторинг уязвимостей

- [`SNYK_SETUP.md`](./SNYK_SETUP.md) - Настройка Snyk для безопасности
  - Интеграция с CI/CD
  - Мониторинг уязвимостей
  - Автоматические проверки

### 🤖 **Интеграции**
- [`TELEGRAM_SETUP.md`](./TELEGRAM_SETUP.md) - Настройка Telegram бота
  - Создание бота через @BotFather
  - Получение токена и Chat ID
  - Интеграция с формой подписки

## 🚀 Быстрый старт

1. **Скопируйте** [`ENV_EXAMPLE.md`](./ENV_EXAMPLE.md) в `.env`
2. **Настройте** Telegram бота по [`TELEGRAM_SETUP.md`](./TELEGRAM_SETUP.md)
3. **Добавьте** ключи аналитики в `.env`
4. **Проверьте** безопасность по [`SECURITY_SETUP.md`](./SECURITY_SETUP.md)

## 📋 Чек-лист настройки

- [ ] Создан файл `.env` на основе `ENV_EXAMPLE.md`
- [ ] Настроен Telegram бот
- [ ] Добавлены ключи аналитики
- [ ] Проверена безопасность
- [ ] Настроен Snyk (опционально)
