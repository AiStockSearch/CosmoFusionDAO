
#### 2. Масштабирование текста
```css
/* Добавить поддержку масштабирования */
html {
  font-size: 16px; /* Базовый размер */
}

/* Использовать относительные единицы */
.text-sm { font-size: 0.875rem; }
.text-lg { font-size: 1.125rem; }
```

#### 3. Фокус видим
```css
/* Улучшить видимость фокуса */
*:focus {
  outline: 2px solid #0ea5e9;
  outline-offset: 2px;
}

/* Для кнопок */
button:focus {
  outline: 2px solid #0ea5e9;
  outline-offset: 2px;
}
```
