# Настройка Telegram бота для формы подписки

## Переменные окружения

Создайте файл `.env` в корне проекта со следующими переменными:

```env
# Telegram Bot Configuration
REACT_APP_TELEGRAM_HTTP_API=your_telegram_bot_token_here
REACT_APP_TELEGRAM_CHAT_ID=your_chat_id_here
```

## Как получить токен бота

1. Найдите бота @BotFather в Telegram
2. Отправьте команду `/newbot` или используйте существующий бот
3. Получите токен в формате: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`

## Как получить Chat ID

### Для личного чата:
1. Отправьте сообщение боту
2. Откройте в браузере: `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates`
3. Найдите `chat_id` в ответе

### Для группы:
1. Добавьте бота в группу
2. Отправьте сообщение в группу
3. Проверьте обновления: `https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates`
4. Найдите `chat_id` группы (обычно отрицательное число)

## Формат сообщений

Бот будет отправлять сообщения в следующем формате:

```
🆕 Новая подписка с сайта CosmoFusion DAO

👤 Имя: John Doe
📧 Email: john@example.com
🌐 Источник: Веб-сайт (форма подписки)
📅 Дата: 18.06.2024, 15:30:45
⏰ Время: 15:30:45
```

## Тестирование

1. Запустите проект: `npm start`
2. Заполните форму подписки
3. Проверьте получение сообщения в Telegram

## Безопасность

- Не коммитьте `.env` файл в репозиторий
- Добавьте `.env` в `.gitignore`
- Используйте разные токены для разработки и продакшена
