# Locales — context.md

**Назначение:**

Папка src/locales/ содержит централизованные файлы переводов для всех языков приложения. Используется для локализации UI и контента.

**Структура:**

- ru.json — переводы на русском
- en.json — переводы на английском

**Принципы работы:**

- Все текстовые ресурсы выносятся в JSON-файлы.
- Ключи переводов согласованы между языками (одинаковая структура).
- Для доступа к строкам используется хук useLocale.
- Автоопределение языка по браузеру/гео, возможность смены вручную.
- Переводы легко редактировать параллельно (одинаковый формат файлов).

**Рекомендации:**

- При добавлении новых ключей — обновлять оба файла.
- Не хранить переводы в компонентах или контентных файлах.
- Для сложных секций использовать вложенные объекты (например, hero.title, hero.subtitle).
