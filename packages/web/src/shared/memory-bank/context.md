# Memory Bank — context.md

**Назначение:**

Memory bank используется исключительно для хранения технического контекста, необходимого для взаимодействия с Cursor (AI/агентом). Не предназначен для хранения пользовательских данных, истории, переводов или бизнес-логики приложения.

**Примеры использования:**
- Хранение информации о реализованных фичах, автотестах, автодокументации.
- Быстрый доступ к состоянию задач, файлов, автогенерируемых описаний.
- Используется только в процессе разработки и взаимодействия с Cursor.

**Важно:**
- Не содержит пользовательских настроек, истории, переводов.
- Не участвует в бизнес-логике приложения.
- Может быть реализован как in-memory storage или через localStorage только для dev-режима.
